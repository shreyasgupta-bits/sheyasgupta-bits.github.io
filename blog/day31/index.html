Using terraform to launch an AWS instance

Terraform which is a popular open source infrastructure automation tool can be used to create and allocate infraturcture and resources easily.

The basic commands of terraform are terraform init, terraform plan and terraform apply.
terraform init is used to initialise the directory where you have saved the .tf file. Terraform installs the provider, in our case aws and downloads it in a hidden sub-directory.
terraform plan prints out an execution plan which is used to go over what changes, additon and deletions your code is going to cause to match your infrastructure to your configuration, and gives you a chance to confirm that everything is according to how you want. terraform apply runs your .tf file and applies the changes.

Terminating an instance using Terraform

Terraform uses a command terraform destroy to delete or terminate an instance that you have created. But say that you have created 2 or more instances using terraform, this command used as is will terminate all of them. To delete only one or a select few, we have to enter and distinguish using a few parameters.
Another was of deleting an instance is to remove the code block from your terrform config file that builds the instance and then using the command terraform apply, Terraform will see that the config file doesn’t have any resources to be allocated whereas your infrastructre does and will promptly terminate that instance.
